# Codebase Index

`.repomap/index.json` is the structure codebase index.
Read before exploring the code — it will save you time. The index auto-updated on commits and pre-commit runs.

# CLI Commands

1. **Find a symbol** 'repomap search <name>'
2. **Refresh the index** 'repomap index'
CLI path: `tools/repomap/repomap.sh` (Unix) or `tools\repomap\repomap.cmd` (Windows).
