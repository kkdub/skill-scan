## Refined Criteria

- `_is_terminal_body([ast.Return(...)])` returns True
- `_is_terminal_body([ast.Raise(...)])` returns True
- `_is_terminal_body([ast.Break()])` returns False (not scope-exit)
- `_is_terminal_body([ast.Continue()])` returns False (not scope-exit)
- `_is_terminal_body([])` returns False (empty body)
- Bare if (no else) returns False even if body terminates
- if/else where both branches terminate returns True
- Nested if/else recursion works to 2+ levels
- for/while body always returns False (might not execute)
- try: if finalbody terminates -> True regardless of other bodies
- match: exhaustive (unguarded wildcard) + all cases terminate -> True
- Conservative: any unrecognized structure -> False
