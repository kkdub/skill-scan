# skill-scan

This repo is a security scanner for agent skills — it detects prompt injection, malicious code, and data exfiltration pre-install.

**Work-in-progress** 
If you stumble onto this repo and use this code, do so at your own risk. (That's the way of things github but if you're new you may not realize.) Skill-scan is better than just reading a skill but certainly not exhaustive. It has no mechanism for checking skills that are auto-updated, a process that should never be enabled.
